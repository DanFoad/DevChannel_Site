Biweekly challenge #2 (UPDATED)

Hello everyone, I'm here to announce the second biweekly challenge!

This week it will be to make a Web App. The goal is to have a file uploader: you choose a file from your pc, upload and it gets stored in the server.

The most basic version of this would be being able to upload text files. After that, display/reproduce the files on the website.

Some improvements would be:

  • more file types
  • drag and drop feature
  • make a better design for the website
  • add restrictions such as file size
  • maximum number of requests
  • etc, the best you can do, try to make it as visible as possible

You are encouraged to make it with teams, and those that do will get extra points for it,

Good luck!

Biweekly challenge #1 (UPDATED)

Hey there! This time the goal will be to create an autocorrect program.

The basic task is to write an autocorrect program.

It should be able to test if a typed word is correct or not and give some feedback to the user.

Improve it as much as you want (graphical interface, analyzing sentences, predicting mistakes... do what you feel that is right, be creative!

If you use a list of words please put it in a file called 'words.txt', and you can use our list!

THIS CHALLENGE IS FOR EVERYONE. Don't be afraid to participate if you know your program will lose, what matters is what you learn :D

FIND A PARTNER or do it on your own, but teams (2-3 people) will have bonus points for cooperation!

Deadline: Sunday, March 20th at mid night (GMT 0) (beginning of Monday :) ) Good luck!

Weekly challenge #3:

Make a todo-bot (a todo-list with which you interact with commands like "add 'make a robot' to bigProjects").

The goal is pretty obvious, you should have the current functionalities:


  • Adding new items
  • Marking items as done
  • Check the list of stuff you still have to do and also the list of completed tasks

Example: Given the command "add 'change player skin'" would add 'change player skin' to the list and "'change player skin' done" would move it to a 'Done' list

You can improve it and simplify it as much as you want by adding functionalities like having multiple lists, reminders, etc..

rules are that you should make it in teams of 2-3 people.

The final projects will be ranked and the best (or maybe a combination of the best) will be integrated by us to the slack chat as bot!

If you want to participate, find 1 or 2 people to team with and start working on it, you can ask me, @daruso , @vaggelisd or @himrin to make you a private channel :) Finally, the submission date is until next Sunday(March 6th) at midnight, you should upload it to a github repo and link us to it in this channel(#weekly_callenge) :D

Good luck!
